                <p>Самая маленькая частица конкретного вещества, которая всё ещё сохраняет его основные свойства, называется молекулой. Молекулы невероятно малы, но даже они состоят из ещё более мелких частиц — атомов.</p>
                <div class="example">Пример: Молекула воды всегда одинакова и состоит из трёх атомов — двух атомов водорода и одного атома кислорода (<span class="formula">H<sub>2</sub>O</span>).</div>

            </div>
            <div class="topic-details">
                <p>Диффузия — это явление, при котором молекулы одного вещества самопроизвольно проникают в промежутки между молекулами другого вещества. Это доказывает, что молекулы непрерывно и беспорядочно движутся.</p>
                <div class="example">Пример: Быстрее всего диффузия происходит в газах (распространение запахов), медленнее — в жидкостях (растворение краски в воде).</div>

                <div class="example">Пример: Если зачистить до блеска два свинцовых цилиндра и плотно прижать их, они прочно сцепятся силами молекулярного притяжения.</div>

                <div class="example">Пример: Лёд (твёрдое тело), вода (жидкость) и водяной пар (газ) состоят из одних и тех же молекул.</div>

    <script>
        function toggleTopic(button) {
            const card = button.closest('.topic-card');
            const details = card.querySelector('.topic-details');
            if (details.style.display === 'block') {
                details.style.display = 'none';
                button.innerText = 'Развернуть';
            } else {
                details.style.display = 'block';
                button.innerText = 'Свернуть';
            }
        }
